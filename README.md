# A-Complex-formation-Model (Branch: Modeling)
Develop a Coarse Grained model for the formation of spliceosome's A-Complex under the influence of antitumoral drugs.

**Note**: This branch contains the funciton related with different model Hypothesis and the notebook in which are analyzed the trends.

**General Idea**: The general idea here is to try different combination of model Hypotesys by generating assumption-constrained random parameters and computing steady states trends for each model. These computation can help us to understand which models better represent the biological process.
